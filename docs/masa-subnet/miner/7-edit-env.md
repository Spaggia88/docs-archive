### Create .env file

In the masa-bittensor directory, create an .env file

```bash
vim .env
```

Add the following variable to your .env file, replacing the URL with your Masa Protocol node's URL

```bash
ORACLE_BASE_URL="http://localhost:8080/api/v1"
```
